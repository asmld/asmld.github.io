Question 3

题目描述：求所有不小于-100的整数，满足加上100后是一个完全平方数，再加上168又是一个完全平方数。

提示：通过数学推导确定数的范围

输出样例：

参考源码：

分析：

  1. 确定范围：

    $$ \begin{split} &n+100=a^2,n+268=b^2\Rightarrow b^2-a^2=168\\ &\Rightarrow b>a\Rightarrow b\geq a+1\\ &168=b^2-a^2\geq (a+1)^2-a^2=2a+1\\ &\Rightarrow a<84 \end{split} $$

  2. 只需通过循环求出所有满足$\sqrt{a^2+168}\in\mathbb{Z}$的 a

  3. $a^2-100$即为要求的 n
